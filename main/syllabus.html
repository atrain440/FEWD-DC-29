Week One - Intro to HTML and CSS

Class 01 - Introduction | HTML

Topics

  • Introduction to GA and Instructional Team
  • Tools and Resources for FEWD
  • HTML Structure (Head and Body)
  • The Document Object Model (DOM)
  • HTML tags
  • Semantic HTML

Class 02 - CSS

Topics

  • Review HTML and Semantics
  • HTML Tables
  • HTML Page Creation Lab
  • CSS Selectors, Properties and Values
  • CSS Properties, What is the good stuff?
  • Linking in HTML Documents
  • Cascading / Inheritance of CSS
  • The Document Object Model, Drawing DOM Trees

Week Two - More CSS

Class 03 - Box Model

Topics

  • Review CSS Properties
  • Explore additional CSS Properties (background, color, font, text-)
  • Link HTML documents
  • Draw DOM Trees
  • Overview of Box Model
  • Explore structural CSS Properties (border, margin, padding)
  • Sizing units (px, %, em, rem)

Class 04 - Layouts

Topics

  • Box Model Review - mini lab
  • Sizing units (px, %, em, rem)
  • CSS Display Property
  • Float and Clear Properties
  • Layout Lab

Week Three - Tooling / Layout Practice

Class 05 - Tooling

Topics

  • Review of Float and Clear
  • File Systems / OS Mods
  • Text Editor Customization
  • Git, Github and your first pull request
  • Layout lab if we have time

Class 06 - CSS Advanced + Layout Lab

Topics

  • CSS Pseudo selectors
  • @font-face
  • Opacity and Transitions
  • Practice making simple webpages with HTML and CSS skills already learned

Week Four - Intro to Programming

Class 07 - Programming

Topics

  • What is a program?
  • Pseudo code as a way to think about logic
  • Recipes and map direction walk-throughs
  • Intro to JS
  • Data Types (Numbers, Strings, Booleans, Null)
  • Basic Ways to Output to Browser
  • Variables
  • Functions

Class 08 - JS Selectors, jQuery, Events

Topics

  • Review programmatic thinking, tips from homework last week
  • Introduce Selectors
  • Introduce Methods
  • Intro to jQuery
  • Events in JS and jQuery - .click(), .val()
  • Lab time

  • Start on Homework in class as group assignment
